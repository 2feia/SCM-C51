### 51单片机红外解码程序.c

> 本程序将HT62222发送的波形用HX13838接收，再使用单片机软解码并将结果显示在8个数码段上。
> 显示内容包括：用户编码8bit2，按键编码bit2

- 程序名：HT62222红外发送，51单片机软解码
- 作者：szy0syz
- 时间：2010.7.18 
- 单片机型号：AT89S52
- 红外接收模块：HX1838
- 红外编码模块：HT6222
